# AcWing 2. 01背包问题
[传送门](https://www.acwing.com/problem/content/2/)

```C++
#include <bits/stdc++.h>
using namespace std;
const int N = 1010;
int f[N], n, m, w, v;

int main() {
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; ++ i) {
        scanf("%d%d", &w, &v);
        for (int j = m; j >= w; -- j) {
            f[j] = max(f[j], f[j - w] + v);
        }
    }
    printf("%d\n", f[m]);
}
```
