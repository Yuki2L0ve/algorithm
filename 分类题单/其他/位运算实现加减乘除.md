# 加法
```C++
int add(int a, int b) {
    int ans = a;
    while (b) {
        ans = a ^ b;    // a ^ b表示a和b无进位相加的结果
        b = (a & b) << 1;   // (a & b) << 1表示a和b相加时的进位信息
        a = ans;
    }
    return ans;
}
```
