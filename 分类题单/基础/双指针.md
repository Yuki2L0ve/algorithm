# 同向双指针
## LC3 无重复字符的最长子串
[传送门](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)
```C++
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        unordered_map<char, int> mp;
        int ans = 0, n = s.size();
        for (int l = 0, r = 0; r < n; ++ r) {
            ++ mp[s[r]];
            while (mp[s[r]] > 1) {
                -- mp[s[l]];
                ++ l;
            }
            ans = max(ans, r - l + 1);
        }
        return ans;
    }
};
```
